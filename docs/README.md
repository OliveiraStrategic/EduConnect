# Documentação

Este diretório reúne toda a documentação do projeto **EduConnect**. Aqui serão armazenados e atualizados os documentos relacionados ao desenvolvimento do sistema, incluindo requisitos, regras de negócio, diagramas, wireframes, estudos de viabilidade e demais artefatos produzidos pela equipe.

A documentação será mantida de forma contínua ao longo do desenvolvimento do projeto, garantindo que as informações estejam sempre organizadas e atualizadas.
